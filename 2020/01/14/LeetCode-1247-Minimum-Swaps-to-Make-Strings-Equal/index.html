LeetCode-1247 Minimum Swaps to Make Strings Equal

发表于 2020-01-14 | 分类于 刷题 ， LeetCode
字数统计: 538 | 阅读时长 ≈ 2

题目描述

You are given two strings s1 and s2 of equal length consisting of letters “x” and “y” only. Your task is to make these two strings equal to each other. You can swap any two characters that belong to different strings, which means: swap s1[i] and s2[j].

样例输入输出

Example 1:

Input: s1 = “xx”, s2 = “yy”
Output: 1
Explanation:
Swap s1[0] > and s2[1], s1 = “yx”, s2 = “yx”.

Example 2:

Input: s1 = “xy”, s2 = “yx”
Output: 2
Explanation:
Swap s1[0] and s2[0], s1 = “yy”, s2 = “xx”.
Swap s1[0] and s2[1], s1 = “xy”, s2 = “xy”.
Note that you can’t swap s1[0] and s1[1] to make s1 equal to “yx”, cause we can only swap chars in different strings.

Example 3:

Input: s1 = “xx”, x2 = “xy”
Output:: -1

解答

首先，如果s1[i] = s2[i], 那么该位置不用作交换

如果s1[i] != s2[i], 那么需要做一次交换使得它们相等。怎么交换呢？
按照贪心的策略, 我们肯定希望这一次交换除了使得i位置的两个字符相等, 也能使得被交换的j位置的两个字符相等.

进一步观察，我们发现，如果

s1[i] = x, s1[j] = x
s2[i] = y, s2[j] = y

那么将s1[i]与s2[j]一次交换便可以使得i、j这两个位置满足相等条件。
同理，如果

s1[i] = y, s1[j] = y
s2[i] = x, s2[j] = x

那么将s1[i]与s2[j]一次交换便可以使得i、j这两个位置满足相等条件。

在其它情况下，做一次交换只能使得一个位置满足相等条件，例如

s1[i] = x, s1[j] = y
s2[i] = y, s2[j] = x

这种情况下，先将s1[i]与s2[i]交换，变成

s1[i] = y, s1[j] = y
s2[i] = x, s2[j] = x

然后再将s1[i]与s2[j]交换

这样思路就很清晰了，我们先去交换那些一次能满足两个位置的情况，再去交换那些一次满足一个位置的情况。
